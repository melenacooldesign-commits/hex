HEX RENDER — мозаика из гексагонов
==================================

Состав:
  index.html  — приложение (открыть в браузере)
  hex.svg     — эталонный контур гексагона из Figma

Как запустить:
  1. Дважды кликните index.html
     или перетащите файл в Chrome / Safari / Firefox
  2. Загрузите изображение (кнопка, drag-and-drop или Ctrl+V)
  3. Настройте размер гексагона, тон, контраст
  4. Скачайте PNG или SVG

Сервер не нужен — всё работает локально в браузере.
