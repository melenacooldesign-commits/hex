# HEX Render

Мозаика из скруглённых гексагонов (статический сайт).

## Локально

Откройте `index.html` в браузере или папку `hex-render/index.html`.

## Vercel

- **Framework Preset:** Other (без сборки)
- **Root Directory:** оставьте пустым (корень репозитория) — `vercel.json` перенаправляет `/` на `hex-render/index.html`
- Либо укажите **Root Directory:** `hex-render` и удалите `vercel.json`

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Приложение (корень, для Vercel) |
| `hex-render/` | Копия проекта (если заливали zip) |
| `hex.svg` | Контур гексагона |
