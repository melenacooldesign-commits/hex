# HEX Render

Мозаика из скруглённых гексагонов (статический сайт).

## Локально

Откройте `index.html` в браузере или папку `hex-render/index.html`.

## Vercel

- **Framework Preset:** Other
- **Root Directory:** пусто (корень репозитория)
- **Build Command:** пусто
- **Output Directory:** пусто (или `.`)
- **Install Command:** можно отключить / оставить пустым
- В корне должны быть `index.html` и `hex.svg` (не в подпапке)

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Приложение (корень, для Vercel) |
| `hex-render/` | Копия проекта (если заливали zip) |
| `hex.svg` | Контур гексагона |
